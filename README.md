# Zomato Restaurant Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data using Python. The objective was to analyze restaurant ratings, pricing, cuisines, customer engagement, and delivery performance to uncover actionable business insights.

---

## Business Objective

The analysis aims to answer key business questions such as:

* Which cuisines are most popular?
* How do ratings vary across restaurants?
* Does higher pricing lead to better ratings?
* What factors influence customer engagement?
* What characteristics define successful restaurants?

---

## Dataset

The dataset contains restaurant-level information including:

* Restaurant Name
* Cuisine Types
* Dining Rating
* Delivery Rating
* Cost for Two
* Number of Votes
* Bestseller Status
* Delivery Availability

Dataset file is available in the `dataset` folder.

## Key Insights

* Restaurant ratings are concentrated around a limited range, indicating moderate customer satisfaction across most restaurants.
* Popular cuisines consistently attract higher customer engagement.
* Customer votes show a positive relationship with restaurant popularity.
* Pricing alone does not guarantee higher ratings.
* Bestseller restaurants generally receive stronger customer engagement.
